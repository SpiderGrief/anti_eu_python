# 👁️ ctOS Chat Control Interceptor & Profiler

Cyberpunk desktop monitor emulator (inspired by Watch Dogs/Buckshot Roulette) created in Python 3.10 as a creative, artistic response to the EU's "Chat Control" surveillance proposals.

---

## ⚡ Features
* **5 Interactive Screens:** Real-time scanning, data interception, a protester manifesto, citizen profiling (press `R`), and a data library (arrow keys).
* **CRT Shader:** Custom NumPy/OpenCV-based retro screen effects.
* **Audio System:** Built-in synthesizer and audio file support.

---

## 🛠️ Installation & Setup (Установка и настройка)

### 1. Download & Unpack
Download and unzip the `ANTI EU.zip` archive into any folder on your computer.

### 2. Install dependencies
Open your terminal/CMD in the unpacked folder and run:
```bash
pip install pygame opencv-python numpy
```

### 3. Add audio files
Place your audio files (`bg_music.mp3`, `page_select.mp3`) in the root folder next to `ctos2.py`.

### 4. Run the app
```bash
python ctos2.py
```

---

## 🎮 Controls
* `1` - `5`: Switch tabs.
* `R`: Scan citizen (Profiler).
* `Left/Right` Arrows: Navigate articles (Database).
* `Esc/Alt+F4`: Exit.

---

## ⚖️ Disclaimer
Artistic simulation/political satire only. No actual data interception.

# 👁️ ctOS Chat Control Interceptor & Profiler (На русском)

Интерактивный киберпанк-эмулятор монитора (в стиле Watch Dogs/Buckshot Roulette) на Python 3.10, созданный как творческий ответ на законы о "Chat Control" в ЕС.

---

## ⚡ Функции
* **5 экранов:** Прослушка, LIVE SCAN (с ИИ-детектором), манифест, профилирование граждан (клавиша `R`) и хакерская база.
* **ЭЛТ-шейдер:** Ретро-эффекты на NumPy/OpenCV.
* **Звук:** Программный синтезатор и поддержка внешних файлов.

---

## 🛠️ Установка и настройка

### 1. Скачивание и распаковка
Просто распакуйте архив `ANTI EU.zip` в любую папку на компьютере.

### 2. Установка зависимостей
Откройте терминал/CMD в папке и выполните:
```bash
pip install pygame opencv-python numpy
```

### 3. Настройка аудио
Добавьте свои файлы `bg_music.mp3` и `page_select.mp3` рядом с `ctos2.py`.

### 4. Запуск
```bash
python ctos2.py
```

---

## 🎮 Управление
* `1` - `5`: Переключение экранов.
* `R`: Сканирование (PROFILER).
